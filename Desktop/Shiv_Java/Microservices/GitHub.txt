LAPTOP to GITHUB:

//Store the user and mail id 
git config --global user.name "Your Name"
git config --global user.email "your_email@example.com"

//Configuraton List
git config --list

//After above get login UI and logged in 


git init

//Add all files
git add .

//Commit the changes (perment save)
git commit -m "Initial commit"


//Connect the GitHub Repository
git remote add origin https://github.com/shivrajkachare/Microservices.git

//Push the changes 
git branch -M main
git push -u origin main

//OR if push the by the branch name 
git checkout -b microservice
git push -u origin microservice




**if the changes in the LAPTOP to GitHub**

//check status --> it shows the changes and info
git status
